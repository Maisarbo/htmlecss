RESUMO DA AULA - INTRODUÇÃO

HTML é uma linguagem de marcação de texto, interpretada por navegadores web, que descreve a estrutura de uma página web, indicando para navegador o conteúdo a ser renderizado para o usuário. É indicado através de seus elementos que são definidos por uma tag inicial, conteúdo e uma tag final.

Pode-se concluir que HTML é uma linguagem de desenvolvimento, ou seja com regras de sintaxe e de semânticas que auxiliam no desenvolvimento de páginas web, para assim terem um padrão e serem interpretadas pelo navegador corretamente como o desenvolvedor gostaria, portanto o navegador não exige que as tags HTML sejam usadas para exibir um texto na tela porém ele não garante que ele será exibido corretamente como o desejado.

Exemplo de elementos HTML

Um exemplo de elementos HTML é <h1>titulo de nivel 1</h1> que é utilizado para indicar em uma pagina titulo de nivel 1, como toda e qualquer tag que tem intuito de indicar algo que será renderizado na tela para usuario deve ser declarado em <body>Exibido na tela</body>.

O elemento <title> </title> é um exemplo de elemento HTML que é declarado em <heade> </heade>. O <title> </title> elemento especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página), além disso tem grande influência no SEO da página pois ele serve como um identificador da página para o google analisar.

                        <!DOCTYPE html> 
                        <html lang="pt-br"> 
                        <heade> 
                        <meta charset="UTF-8"> 
                        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                        <title>Título exibido na barra de navegação ou na guia da página</title>
                        </heade> 
                        <body> 
                           <h1>titulo de nivel 1</h1> 
                           qualquer texto sem marcação... 
                        </body>
