RESUMO DA AULA - LISTAS E LINKS

O que são listas?

Listas é um recurso em HTML que permite exemplificar um texto através da criação de tópicos. Um exemplo comum de uma utilização de listas é na criação de menus de sites que são uma lista não ordenada de links para navegação.

Exemplos dos três tipos de listas

Listas não ordenadas

Cujo itens das listas não têm necessidade de terem marcadores que indicam sua ordem.

                        <!DOCTYPE html> 
                        <html lang="pt-br"> 
                        <heade> 
                        <meta charset="UTF-8"> 
                        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                        <title>Título exibido na barra de navegação ou na guia da página</title>
                        </heade> 
                        <body> 
                           <h2> Ingredientes </h2> 
                           <ul> 
                                <li> 1col de açucar </li> 
                                <li> 1col de café   </li>
                                <li> 1 copo de água </li>
                           
                           </ul> 
                           
                        </body>

Listas ordenadas

Cujo itens das listas têm necessidade de terem marcadores que indicam sua ordem.

                        <!DOCTYPE html> 
                        <html lang="pt-br"> 
                        <heade> 
                        <meta charset="UTF-8"> 
                        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                        <title>Título exibido na barra de navegação ou na guia da página</title>
                        </heade> 
                        <body> 
                           <h2> Ordem de chegada </h2> 
                           <ul> 
                                <li> Paula  </li> 
                                <li> Vitor  </li>
                                <li> Paulo </li>
                           
                           </ul> 
                           
                        </body>

Listas de definições

Cujo itens das listas tem a necessidade de terem uma descrição mais especifica.

                        <!DOCTYPE html> 
                        <html lang="pt-br"> 
                        <heade> 
                        <meta charset="UTF-8"> 
                        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                        <title>Título exibido na barra de navegação ou na guia da página</title>
                        </heade> 
                        <body> 
                           <h2> Capital dos países </h2> 
                           <dl> 
                                <dt> Brasil </dt> 
                                <dd> Brasília </dd> 
                                <dt> França  </dt>
                                <dd> Paris </dd> 
                                <dt> Uruguai </dt>
                                <dd> Montevidéu </dd> 
                           
                           </dl> 
                           
                        </body>

Além destas formas de marcação de listas pode se aninhar mais de um tipo de lista em outra lista.

O que são Web Sites?

Um WEB site é uma ou conjunto de páginas WEB, que são hipertextos, construídas utilizando linguagem de marcação HTML que são interligadas através de hiperlinks.

O que são links?

Links podem ser utilizados de diversas formas em uma página WEB, um link basicamente tem como funcionalidade encaminhar a exibição de outro documento quando um usuário clicar nele, à parte visível ao usuário para ser clicada pode ser tanto um texto ou imagem, sendo considerado um hiperlink.

Exemplo das três formas de utilização de links

                        <!DOCTYPE html> 
                        <html lang="pt-br"> 
                        <heade> 
                        <meta charset="UTF-8"> 
                        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                        <title>Título exibido na barra de navegação ou na guia da página</title>
                        </heade> 
                        <body> 
                           <h2> Link externo que interligam com outros sites. </h2> 
                           <a href="https://ebaconline.com.br/">EBAC </a> 
                           <h2> Link internos que interligam com outro local da pagina. </h2> 
                           <a href="#top">Volta ao topo</a>
                           <h2>Link internos que interligam com outra pagina do mesmo site. </h2> 
                           <a href="#lista-nao-ordenada">Ver a lista não ordenada</a>
                        </body>
