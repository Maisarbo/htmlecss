RESUMO DA AULA - FORMATAÇÃO DE TEXTO

Como HTML é uma linguagem de marcação, sua função é indicar para o navegador como um hipertexto deve ser exibido para o usuário, portanto existem diferentes formas, entre elas temos os títulos que descrevem a essência do texto de uma página, em HTML comumente existem seis níveis de título diferentes.

                        <!DOCTYPE html> 
                        <html lang="pt-br"> 
                        <heade> 
                        <meta charset="UTF-8"> 
                        <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                        <title>Título exibido na barra de navegação ou na guia da página</title>
                        </heade> 
                        <body> 
                           
                           <h1> NÍVEL 1 </h1> 
                           <h2> NÍVEL 2 </h2> 
                           <h3> NÍVEL 3 </h3> 
                           <h4> NÍVEL 4 </h4> 
                           <h5> NÍVEL 5 </h5> 
                           <h6> NÍVEL 6 </h6> 
                    
                        </body>

A linguagem HTML tem intuito de apenas indicar como texto será exibido de forma semântica, ou seja, atribuindo significados para o texto, quando se refere ao nível do título se refere a sua posição semântica no texto não tamanho. Tudo que se refere à estilo é determinado pelo CSS.

HTML permite você atribuir significados estilizando um texto

Estilização de textos somente com HTML

Para o desenvolvimento web é necessário utilizar boas práticas de desenvolvimento, como por exemplo utilizar tags HTML somente para atribuir sentido a um texto, mas para o desenvolvimento de layouts para um servidor de e-mail não é possível utilizar o HTML somente para uso semântico pois os recursos de CSS são limitados, portanto há possibilidade de usar tags em HTML somente para estilização mesmo não sendo recomendada para desenvolvimento web. 

            <!DOCTYPE html> 
            <html lang="pt-br"> 
            <heade> 
            <meta charset="UTF-8"> 
            <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
            <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
            <title>Título exibido na barra de navegação ou na guia da página</title>
            </heade> 
            <body> 
               
               <h1> NÍVEL 1 </h1> 
               <b>Texto em negrito</b> 
               <i>Texto em itálico</i> 
               
        
            </body>
