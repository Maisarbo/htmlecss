RESUMO DA AULA - TABELAS

São estruturas de organização de dados, em linhas e colunas. Inicialmente no desenvolvimento web os layouts eram feitos em tabelas utilizando recursos HTML pois os recursos eram limitados. Atualmente desenvolvimento de layouts em tabelas HTML é utilizado somente para servidor de e-mails em que interpretador CSS é limitado.

Exemplo de tabela

			   <!DOCTYPE html> 
			   <html lang="pt-br"> 
			   <heade> 
			   <meta charset="UTF-8"> 
			   <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
			   <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
			   <title>Título</title>
			   </heade> 
			   <body> 
					   
				  <h1> NÍVEL 1 </h1> 
				  <table border="1" width="100%">
					<thead>
					  <tr>
						<th>Coluna 1</th> 
						<th>Coluna 2</th> 
						<th>Coluna 3</th> 
					  </tr> 
					</thead>
					<tbody>	
					  <tr align="center">
					    <td>1A</td> 
					    <td>1B</td> 
					    <td>1C</td> 
				        </tr> 
					   <tr align="right">
					    <td>2A</td> 
					    <td>2B</td> 
					    <td>3C</td> 
				       </tr> 
					   <tr>
					    <td>3A</td> 
					    <td colspan="2">3BC</td> 
				       </tr> 
				</tbody>
				<tfoot>
					   <tr>
					    <td colspan="3">Rodapé da tabela</td> 
					   </tr>
				</tfoot>
				</table>

Tabela como layout

			  <!DOCTYPE html> 
			  <html lang="pt-br"> 
			  <heade> 
			  <meta charset="UTF-8"> 
			  <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
			  <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
			  <title>Título exibido na barra de navegação ou na guia da página</title>
			  </heade> 
			  <body> 
			  <h2> Tabela como layout </h2> 
	               <table width="600">
		           <tr height="90">
			           <td colspan="4" bgcolor="#BBCCEE" align="center"> Cabeçalho</td>
		           </tr>
		           <tr height="300">
			           <td width="20" bgcolor="#0000FF"></td>
					   <td width="60%" bgcolor="#DDDDDD">
					   <h2>Título</h2>
					   <p>Um exemplo de parágrafo</p>
					   </td>
			           <td bgcolor="#EEEEEE">Coluna 2</td>
			           <td width="20"bgcolor="#0000FF"></td>
		           </tr>
		           <tr>
					<td bgcolor="#FF0000" colspan="4" align="center">Rodapé</td>
		           </tr>
	               </table>
				</body>
