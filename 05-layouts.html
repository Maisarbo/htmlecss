RESUMO DA AULA - LAYOUTS

Layout é forma de organização dos elementos gráficos em HTML

Atualmente é possível desenvolver layouts com auxílio de elementos semânticos HTML, que definem diferentes partes do texto.

                            <!DOCTYPE html> 
                            <html lang="pt-br"> 
                            <heade> 
                            <meta charset="UTF-8"> 
                            <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
                            <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                            <title>Título exibido na barra de navegação ou na guia da página</title>
                            </heade> 
                            <body> 
                             <h2> Ordem de chegada </h2> 
                             <section> 
                                <nav> 
                                    <ul> 
                                        <li></a href="#home">Home</a></li> 
                                        <li></a href="#sobre">Sobre</a></li>
                                        <li></a href="#contato">Contato</a></li> 
                                        </ul> 
                                </nav>

                                    <h1> Título Artigo</h1>
